## What Gear Slots Do

**Training program biases.** Equipping a gear item adjusts the workout program's emphasis. Boxer's sleeves → rotational power, speed, shoulder endurance. Gymnast's sleeves → rings, handstand press, muscle-up progressions. Powerlifter's belt → heavy hip hinge, squat, deadlift curves.

**Video game classes.** A gear SET = a build = a class. Powerlifter: heavy belt + lifting shoes + chalk hands + basic tunic. Swimmer: light sleeves + bare feet + streamline tunic. CrossFit: all-rounder set with moderate everything. Users mix and match — no locked classes, just loadout combinations.

**Exercise library filter.** Every exercise in the 2,085 library maps to one or more gear slots. Tap a slot → see all exercises that train through that body zone. This organizes the entire library into 7 body-zone categories without replacing the existing movement pattern taxonomy.

**Base-7 octave progression.** Gear items level up like video game items. A sleeve item at octave 1.3 = three tiers of Ionic progression in that training bias. Leveling up unlocks harder exercises within that slot's family.

**Training arcs, not daily changes.** Gear sets fit to programs and mesocycles. You might wear the same set for 4-8 weeks. The system can default-rotate gear with the rotation engine if the user doesn't actively manage it. Gear molts with training, like the planned layer molts to actual.
